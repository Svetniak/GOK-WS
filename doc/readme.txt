This Archive will have the whole instructions to create a blog app, that will help to create and deploy articles on the site, and manage users and Administrators, using "Flask and MariaDB".

Most of the descibes in this documents comes from the online tutorial of flask: https://flask.palletsprojects.com/en/2.2.x/tutorial/ and adjusted to the particular use for this application, transcriped and translated to spanish for better understanding.

To follow the tutorial and undestand correctly how it is working, you neet to read the tutorial.txt file, along with the files described in it, as you write your own files.
In the code Files you will find numeric marks, this are references from the tutorial.txt, where is described what it does.

This material will also be posted in the app resulting from this berry tutorial.
